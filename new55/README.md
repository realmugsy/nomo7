# Игровой Портал - Нонограммы (Nonogram Style)

Простая браузерная игра "Нонограммы" (Японские кроссворды), написанная на чистом HTML, CSS и JavaScript.

## Особенности и Функционал

*   **Рисование Canvas**: Игровое поле отрисовывается на элементе `<canvas>`, что обеспечивает гибкость и высокую производительность рисования.
*   **Локализация (i18n)**: 
    *   Полная поддержка переключения языков (Русский / English).
    *   Тексты хранятся в отдельных JSON-файлах (`lang/ru.json`, `lang/en.json`).
    *   Язык переключается мгновенно без перезагрузки страницы.
    *   Выбранный язык сохраняется в `localStorage`.
*   **Адаптивный дизайн**: Интерфейс оптимизирован как для больших экранов, так и для мобильных устройств. Используется CSS Grid и Flexbox.
*   **Чистая архитектура**: Четкое разделение ответственности между HTML, CSS и модулями JavaScript.

## Структура проекта

```text
/
├── index.html          # Точка входа, основная разметка
├── css/
│   └── style.css       # Все стили оформления (переменные, макет, адаптивность)
├── js/
│   ├── script.js       # Логика игры (Canvas, обработка ввода)
│   └── i18n.js         # Модуль локализации (загрузка JSON, обновление DOM)
└── lang/
    ├── ru.json         # Русский перевод
    └── en.json         # Английский перевод (English translation)
```

## Как запустить

### Вариант 1 (Рекомендуемый): Локальный сервер
Так как проект использует `fetch` для загрузки JSON-файлов локализации, браузеры могут блокировать эти запросы при открытии файла напрямую (CORS Policy). Рекомендуется использовать простой локальный сервер.

Если у вас установлен **Python**:
```bash
python -m http.server 8000
```
Затем откройте в браузере: `http://localhost:8000`

### Вариант 2: Прямое открытие
Вы можете просто открыть файл `index.html` в браузере.
*Примечание: В некоторых браузерах (например, Chrome) переключение языков может не сработать без флага `--allow-file-access-from-files` или без использования локального сервера.*

## Технологии

*   **HTML5**: Семантическая верстка.
*   **CSS3**: CSS Variables, Flexbox, Grid Layout.
*   **JavaScript (ES6+)**: `async/await`, `Modules` (структурно), `Canvas API`, `localStorage`, `fetch API`.
