# Nonogram World

**Nonogram World** is a free, modern, and interactive online logic puzzle game (also known as Nonograms, Griddlers, or Japanese Crosswords).

## ðŸ§© About the Game

Solve infinite procedurally generated nonogram puzzles! The game runs entirely in your browser and offers a seamless experience on both desktop and mobile devices.

Unlike traditional static puzzle sites, Nonogram World generates unique puzzles on the fly using a deterministic seed system, ensuring you never run out of challenges.

## âœ¨ Features

*   **Infinite Puzzles:** Play standard 5x5, 10x10, 15x15, or 20x20 grids.
*   **Daily Challenges:** A unique, same-for-everyone puzzle every day. Compete with friends on the same board!
*   **Difficulty Levels:** ranges from Very Easy to Very Hard.
*   **Smart Tools:**
    *   **Fill / Cross:** Standard tools for solving.
    *   **Auto-Cross:** Automatically marks empty rows/columns (optional).
    *   **Hints:** Verify rows/columns or reveal cells if you get stuck.
    *   **Undo/Redo:** Experiment without fear.
*   **Expedition Mode:** (Coming Soon) A campaign mode map. Currently features an email subscription for updates.
*   **Localization:** Fully translated into English and Russian.
*   **Theme Support:** Dark and Light modes.
*   **PWA Support:** Installable as an app on your device.
